
# ttk火车票查询工具
***
####功能说明:
    本程序提供了命令行火车票查询功能,以表格的形式显示查询
    的结果.
####使用说明:
    

火车票查询工具:
     本程序参考自https://github.com/protream/iquery,在此基础
上做了一些改进:
             1.从py3.x改成了py2.x
             2.可以支持票价查询
             3.windows下支持简体中文
	     4.程序中编码统一为GB2312
